# SPECS: show-unit-definition

## Purpose

Define the application DTO and command-builder behavior for showing a unit
definition.

## Origin

Source use case: docs/requirements/use-cases/uc-show-unit-definition.md

## Acceptance Criteria

- Table and flow viewers consume the same normalized
  `absolutePath -> UnitDefinitionDialogDto` mapping.
- Supported `ajs` commands are generated by application helpers, not assembled
  directly in presentation code.
- Desktop and web dialog-open behavior remains stable.

## Implementation Notes

- Keep command support aligned with the JP1/AJS v13 command-reference feature.
