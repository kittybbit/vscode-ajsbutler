# Glossary

## AJS Definition

Raw JP1/AJS3 definition text.

## Unit

A normalized domain concept representing a JP1/AJS element.

## Unit List

Tabular representation of normalized units.

## Flow Graph

Graph-oriented representation of unit relationships.

## Parser Output

Raw or near-raw structure generated from grammar parsing.

## Normalized AJS Model

Stable internal representation independent from parser and UI library details.
